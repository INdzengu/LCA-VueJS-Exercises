Cooking Masterclass Catalogue
Project Overview
This project is a Vue.js application developed for the "Cooking Masterclass" platform. It allows users to browse a catalogue of cooking courses, view details such as the chef and price, and manage a personal wishlist.

Key Features:

Dynamic course listing using component-based architecture.

Real-time wishlist counting via event emission.

Interactive filtering to view only available courses.

Responsive, modern UI with hover animations and price formatting.
